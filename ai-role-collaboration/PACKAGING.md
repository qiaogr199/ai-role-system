# Portable AI Role System

Generic skills (copy to any repo):

```
.agents/skills/
  ai-role-collaboration/
  role-init/
  role-pm/ role-ui/ role-arch/ role-dev/ role-qa/ role-review/ role-ops/
.cursor/rules/
  role-router.md          # or generate from role-init template
```

Per-repo (generated by **role-init**):

```
.agents/project/role-project.md   # binding manifest
AGENTS.md                         # module map
doc/ai-handoffs/_templates/
doc/ai-handoffs/_index.md
```

## Quick start in a new repo

1. Copy generic skills + optional role-router
2. Invoke skill **`role-init`**
3. `/pm your feature`

## Design rule

**Skills = generic. `role-project.md` = all project facts.**

Never fork role-* skills for project-specific red lines — edit `role-project.md` instead.
